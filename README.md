### 基于free-claude-code增加了对阿里百炼免费模型的支持
  - 阿里对新用户免费试用100多个模型，但是每个模型都有限制用量。
  - 此项目针对每个模型限制用量增加了自动切换机制，当一个模型使用达到用量，继续切换下一个模型，用户无感知，达到免费额度使用的最大化。
  - 启动方式见官方文档。
